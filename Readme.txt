Readme Magento Storeloc by Sutrisno Yao 17-8-2016

1. Put the folder 'app/design..' contents to the correct theme folder you have.
2. Put all your stores data in 'vivere_store_location' table, including images path and longitude-latitude data.
3. You can find the longitude and latitude data by using this site : http://mondeca.com/index.php/en/any-place-en
4. You can always modify the template from ..template/vivere_store/store.phtml
5. Please use it at your own risk
